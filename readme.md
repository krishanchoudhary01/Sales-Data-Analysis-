# 📊 Sales Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing a sales dataset to extract meaningful business insights. The analysis includes data cleaning, exploratory data analysis (EDA), and data visualization.

---

## 🎯 Objective

* Understand sales patterns and trends
* Identify top-performing regions, products, and customers
* Analyze seasonal sales behavior

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Power BI

---

## 🧹 Data Cleaning

* Handled missing values
* Converted date columns to datetime format
* Removed duplicates
* Created new features like Year and Month

---

## 📊 Key Analysis Performed

* Sales by Region
* Sales by Category
* Monthly Sales Trends
* Top 10 Products
* Top Customers
* Sales Distribution
* Correlation Analysis

---

## 📈 Visualizations

* Line charts for trends
* Bar charts for comparisons
* Heatmaps for relationships
* Histogram for distribution

---

## 💡 Key Insights

* Certain regions contribute the highest sales
* Sales show seasonal patterns across months
* A few products drive the majority of revenue
* Customer contribution is uneven (top customers dominate)

---

## 📊 Dashboard

An interactive dashboard was created using Power BI to visualize:

* Total Sales
* Sales by Region
* Monthly Trends
* Category-wise performance

---

## 🚀 Conclusion

This project demonstrates end-to-end data analysis:

* Data Cleaning
* Feature Engineering
* Data Visualization
* Business Insights
